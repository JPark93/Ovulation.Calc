READ ME!
Dataset should be any matrix or data frame objet
  Data should be formatted as follows:
    Column 1 - Survey Date (SD)
    Column 2 - Period Before Last (PBL)
    Column 3 - Last Period (LP)
    Column 4 - Next Period (NP)
      If you do not follow this simple format. Reformat your data.
  date.format should be either "USA" or "WORLD"
    USA = MM/DD/YYY
    WORLD = DD/MM/YYY
